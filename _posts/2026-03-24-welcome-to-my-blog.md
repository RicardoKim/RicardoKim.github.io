# 블로그를 시작하며

안녕하세요! 개인 기술 블로그를 시작하게 되었습니다.

## 이 블로그에서 다룰 내용

이 블로그에서는 다음과 같은 주제들을 다룰 예정입니다:

- **백엔드 개발**: 서버, API 설계, 데이터베이스 최적화
- **시스템 아키텍처**: 분산 시스템, 마이크로서비스, 클라우드
- **DevOps**: CI/CD, 컨테이너, 인프라 자동화
- **문제 해결**: 실무에서 만난 이슈와 해결 방법

## 블로그 기술 스택

이 블로그는 다음 기술로 구축되어 있습니다:

- **Jekyll**: 정적 사이트 생성기
- **Chirpy 테마**: 기술 블로그에 최적화된 Jekyll 테마
- **GitHub Pages**: 무료 호스팅
- **Utterances**: GitHub Issues 기반 댓글 시스템

앞으로 꾸준히 배운 것들을 기록해 나가겠습니다. 잘 부탁드립니다!
